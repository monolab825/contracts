# Contributing to Orbit Protocol

We warmly welcome contributions from the community. Whether you're fixing bugs, adding new features, or improving documentation, your help is appreciated. Here's how you can contribute:

## Getting Started

1. **Fork the repository**: Start by forking the Orbit Protocol repository to your GitHub account.
2. **Clone your fork**: Clone your forked repository to your local machine for development.

    ```
    git clone https://github.com/orbit-protocol/contracts.git
    cd contracts
    ```

3. **Set up your environment**: Follow the setup instructions in the README.md to set up your development environment.

## Making Changes

1. **Create a branch**: Create a new branch for your changes. Use a descriptive name that reflects the nature of your contribution.

    ```
    git checkout -b feature/your-feature-name
    ```

2. **Make your changes**: Implement your changes, making sure you follow the coding standards and guidelines of the project.

3. **Test your changes**: Ensure your changes do not break existing functionality and that all tests pass.

4. **Commit your changes**: Write clear and concise commit messages that explain the nature of your changes.

    ```
    git commit -m "Add a descriptive message explaining your changes"
    ```

## Submitting a Pull Request

1. **Push your changes**: Push your changes to your forked repository on GitHub.

    ```
    git push origin feature/your-feature-name
    ```

2. **Create a pull request**: Go to the original Orbit Protocol repository and create a new pull request from your feature branch. Fill in the pull request template with all the relevant information.

3. **Review and discussion**: Your pull request will be reviewed by the maintainers, who may provide feedback or request changes. Engage in any discussions to clarify or improve your contribution.

## Code of Conduct

By participating in this project, you agree to abide by its [Code of Conduct](CODE_OF_CONDUCT.md). Respectful and constructive communication is expected at all times.

Thank you for contributing to the Orbit Protocol. Your efforts help us build a stronger and more inclusive DeFi ecosystem!
